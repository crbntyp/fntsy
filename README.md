# fntsy

A minimal Fantasy Premier League stats viewer with live gameweek data.

## Features

- **Live Points Tracking** - Real-time player points during active gameweeks
- **Top Players by Position** - View top 10 players sorted by various metrics
- **Team Rosters** - Browse full squad for any Premier League team
- **My Team** - Enter your FPL Team ID to view your squad on a pitch layout
- **Results Ticker** - Live scrolling match results and fixtures

## Sorting Options

- Live/GW Points
- Total Points
- Price
- Ownership %
- Transfers In/Out

## Tech Stack

- React 18
- Vite
- FPL API (via proxy)

## Development

```bash
npm install
npm run dev
```
